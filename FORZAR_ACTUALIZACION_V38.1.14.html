Mejoras importantes en el sistema de historias clínicas

📋 Mejoras Implementadas

  • Validación de HC Única: Ahora el sistema valida que un paciente solo pueda tener una historia clínica activa
  • Flujo Mejorado: Al seleccionar un cliente con HC activa, se muestra automáticamente el modal para agregar nueva admisión
  • Corrección de Race Condition: Solucionado el problema de números duplicados en historias clínicas
  • Retry Logic: El sistema ahora reintenta automáticamente en caso de conflictos de numeración
  • Feedback Visual: Alertas claras cuando un paciente ya tiene HC activa
⚠️ Acción Requerida
Para aplicar esta actualización correctamente, es necesario limpiar la caché del navegador. Sigue los pasos a continuación para asegurar que veas la versión más reciente.

Pasos para Actualizar:

  1. Presiona Ctrl + Shift + Delete (Windows) o Cmd + Shift + Delete (Mac)
  2. Selecciona "Imágenes y archivos en caché"
  3. Haz clic en "Borrar datos"
  4. Recarga la página con Ctrl + F5 (Windows) o Cmd + Shift + R (Mac)
