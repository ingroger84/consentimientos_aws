## ✅ Checklist de Uso

```
PARA CLIENTES:
□ Abrir index-tutoriales.html
□ Clic en "Tutorial Interactivo"
□ Enviar por email o publicar en web

PARA CAPACITACIONES:
□ Abrir tutorial-pago-presentacion.html
□ Presionar F11 para pantalla completa
□ Navegar con flechas del teclado

PARA DOCUMENTACIÓN:
□ Abrir TUTORIAL_PAGO_FACTURA.md
□ Leer o copiar contenido
□ Publicar en base de conocimiento

PARA CREAR PDF:
□ Leer GUIA_CREACION_TUTORIAL_PDF_VIDEO.md
□ Tomar capturas de pantalla
□ Usar Pandoc o herramienta similar

PARA CREAR VIDEO:
□ Leer GUIA_CREACION_TUTORIAL_PDF_VIDEO.md
□ Instalar OBS Studio o Loom
□ Grabar siguiendo el guion
```
