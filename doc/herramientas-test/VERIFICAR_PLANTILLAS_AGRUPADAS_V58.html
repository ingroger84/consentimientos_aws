💡 Soluciones Comunes

Si no ves los cambios:

1. Hard Refresh: Ctrl+Shift+R (Windows/Linux) o Cmd+Shift+R (Mac)
2. Borrar cache del navegador:
- Chrome: Settings → Privacy → Clear browsing data
- Firefox: Settings → Privacy → Clear Data
3. Probar en modo incógnito/privado
4. Verificar que estás usando usuario Super Admin (sin tenant)
5. Verificar en Network tab que los archivos JS tienen código 200 (no 304)
6. Verificar que version.json muestra 41.1.6
