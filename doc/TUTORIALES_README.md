## 🚀 Inicio Rápido

### Opción 1: Índice HTML (Recomendado)
```
Abrir: index-tutoriales.html
```
Página de inicio con acceso a todos los materiales.

### Opción 2: Script PowerShell
```powershell
.\abrir-tutoriales.ps1
```
Menú interactivo para seleccionar qué tutorial abrir.

- **Para clientes**: `tutorial-pago-interactivo.html`
- **Para presentar**: `tutorial-pago-presentacion.html`
